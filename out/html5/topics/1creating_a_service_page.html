Creating a Service Page

Create and publish a service page.

  1. Sign in to the Drupal website
  2. In the top menu, hover over Content then Add content then Service
  3. Fill in the content specified below (Service page template)
  4. Add a note to the Revision log message on the right side of the page
    • Briefly describe what you did to the page
  5. To publish the page, check the box next to Published and Save
    • To save the page as a draft to publish later, leave the box unchecked and Save
